Introduction

Within you will find code examples of how to use Wall of Coins web widgets. For the more advanced users, within also contains the Wall of Coins REST API. Everyone is welcome to integrate the Wall of Coins platform into their apps in any way they deem useful to the digital currency community.

Web Widget

Web Widget

The web widget allows your web site to use Wall of Coins' Buying Wizard functionality. There are many use cases for this, but the most common is to give your end users an easy way to acquire bitcoin to use on your web site!

Buying Bitcoin API

Buying Bitcoin API

If you need more advanced functionality, or if you are integrating the Wall of Coins platform into your custom application, the Buying API is exactly what you want your developers to look into. We include a bare reference client, and the Wall of Coins website (https://wallofcoins.com) uses this API entirely.
